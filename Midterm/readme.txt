Package Dependencies:

1. Tweet-Preprocessor:  Preprocessor is a preprocessing library for tweet data written in Python. This library makes it easy to clean, parse or tokenize the tweets. Install using: $ pip install tweet-preprocessor

2. TextBlob: NLP library to perform tasks like sentiment analysis. Install using: $ pip install textblob

3. Tweepy: Library for accessing the Twitter API. Install using: $ pip install tweepy

4. NLTK: NLP library. Install using: $ pip install nltk
